Project 1

This project had the goal to get me accustomed to Ansible structures and playbooks.
Playbooks:
    1.pull running config and generate a backup file that is also timestamped.
    2.create loopbacks on multiple devices using yml variables
    3.pull version and inventory info.

Project 2
In this project I learned about separating playbooks into tasks. Using jinja2 templates I create a functional 3 tier campus design via Ansible.

Project 3
In this project I use ansible to audit Radius NTP and Banner info on multiple devices. If audit fails remediation is applied and audit is ran again.
Aggregate and per-device reports are generated after every audit.
